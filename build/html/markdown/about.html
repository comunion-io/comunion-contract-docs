如何搭建 solidity 环境

how to start

  1. install truffle

npm install -g truffle

  1. init project

truffle init

how to created local environment

  1. use npm install remixed tools

 npm install -g @remix-project/remixd

  1. open remixed page ( http://remix.ethereum.org)

  2. start in your source code, run

remixed -s [your source code list] --remix-ide http://remix.ethereum.org

  1. open “https://remix.ethereum.org/#optimize=true&runs=200”

  2. click home -> file -> connect to local host;

  3. after loading, happy coding

  4. if you installed ganache, you can use ganache;
